# 📌 Post Feed Page (Frontend Project)

## 🚀 Overview

This project is a simple and responsive Post Feed Page built using HTML, CSS, and Bootstrap. It displays multiple posts in a clean layout and provides a basic structure for social media or blog-style applications.

---

## ✨ Features

* 📄 Display multiple posts
* 🎨 Clean and responsive UI
* ⚡ Built with Bootstrap for fast styling
* 🔘 Interactive buttons (View Post / Logout)
* 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5

---

## 📂 Project Structure

* `index.html` → Main page
* `feedpage.css` → Custom styles

---
